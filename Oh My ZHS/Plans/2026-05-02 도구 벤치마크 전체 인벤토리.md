# 도구 벤치마크 전체 인벤토리

> getin.kr, vivoldi.com/tools, googsu.com, itool.co.kr의 도구를 sitemap/카테고리/내부 링크 기준으로 재조사해 ZHS 도구 제작 후보군을 정리한 노트.

## 조사 방식

- getin.kr: `/sitemap.xml`에서 `/tools/**` 전체 수집
- vivoldi.com: sitemap index를 따라 `/tools/**` 수집
- googsu.com: sitemap이 HTML로 반환되어 홈/카테고리 내부 링크 크롤링
- itool.co.kr: sitemap index + RSS 확인. PDF/이미지/텍스트 도구, 양식, 샘플 파일 페이지를 분리

## 수집량 요약

- getin.kr: 게임 포함 135개 도구 URL
- vivoldi.com/tools: 19개 도구
- googsu.com: 28개 도구
- itool.co.kr: 원시 sitemap/link 기준 1,030개 URL
  - PDF/이미지/텍스트 도구 약 28개
  - 문서 양식 30개
  - 샘플 파일 페이지 38개 수준
- 정규화 후 공개 도구 후보 distinct: 약 173개

## 이미 ZHS 구현/플랜에 반영된 도구

- 아이콘/파비콘 생성기
- 이미지 포맷 변환
- PDF Toolkit
- OG 이미지/썸네일 생성기
- QR/Barcode 생성기
- 웹훅 요청 시뮬레이터
- 내 IP 확인

## 우선 제작 Batch 1 — 기본 유입/실사용 도구

1. JSON/YAML/XML 통합 포매터·검증·뷰어
2. 글자수/단어수/줄수/바이트 카운터
3. Base64 인코딩/디코딩
4. URL 인코딩/디코딩
5. 텍스트 비교/diff
6. JWT 디코더
7. 해시 생성기
8. UUID 생성기
9. 정규식 테스트/예제

## Batch 2 — 이미지/디자인 워크스페이스 확장

1. 이미지 압축
2. 이미지 리사이즈/크롭/회전
3. EXIF/이미지 메타데이터 보기·제거
4. 워터마크 추가
5. 모자이크/블러 처리
6. 색상 선택기 + 이미지 색상 추출/팔레트
7. HEIC 뷰어/변환 — 기술 검증 후
8. SVG 자르기/최적화 — 기술 검증 후

## Batch 3 — 문서/PDF 확장

1. PDF 텍스트 추출
2. PDF 자르기
3. PDF 페이지 삭제
4. PDF 2쪽을 1쪽으로 병합
5. EPUB to TXT
6. 엑셀 병합

PDF 관련 기능은 새 메뉴를 늘리기보다 기존 `/tools/pdf-toolkit` 내부 탭으로 확장하는 방향이 좋다.

## Batch 4 — 생활/계산/시간 도구

1. 단위 변환 통합 고도화
2. 환율 계산기 live-rate 고도화
3. 날짜 계산기
4. D-Day 계산기
5. 카운트다운
6. 스톱워치
7. 포모도로
8. 퍼센트/부가세/팁 계산기
9. 대출/급여/퇴직금/연차 계산기

한국 법/세금 관련 계산은 고지, 산식 출처, 업데이트 날짜를 포함해야 한다.

## 보류 또는 분리할 항목

- DNS Lookup, CIDR, TLS, 방화벽 체크, 검색엔진 봇 IP 검사
  - 현재 ZHS 네트워크 도구는 사용자 결정에 따라 `내 IP 확인 / Check My IP` 단일 기능 유지
- 미니게임/운세/사주/타로
  - 메인 도구 허브보다 experiments/product layer로 분리 검토
- 법률 문서양식 다운로드
  - 책임/업데이트 부담이 커서 현재는 직접 제공보다 체크리스트/작성 가이드 형태가 안전
- 웹사이트 화면 캡처, 웹사이트 이미지 추출, 유튜브 썸네일 다운로드
  - 서버/CORS/저작권 이슈 검토 후 진행
